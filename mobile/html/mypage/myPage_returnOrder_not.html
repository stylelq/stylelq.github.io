취소가능

  • 입금대기 : 주문취소가 가능합니다.
  • 결제완료 : 주문취소가 가능합니다. 결제수단별 환불안내를 확인해주시기 바랍니다.
  • 상품준비중 : 배송을 위하여 물류/매장으로 주문정보가 전달되어 주문취소가 불가능합니다.
  • 배송중 : 택배사가 배송을 시작하여 주문취소가 불가능합니다.
  • 배송완료 : 배송이 완료된 상태이므로 반품 메뉴를 이용하시기 바랍니다.

결제수단별 환불안내

신용카드

  • 승인취소 : 신용카드 승인취소는 매입이 되기 전이기 때문에 승인한 당일 취소됩니다.
  • 매입취소 : 카드사로 매입취소 신청을 하면 카드사에서 취소처리를 하는데 최대 1~2주 정도 소요 됩니다.
    • 카드사별 취소 반영이 차이가 있습니다.
    • 각 카드사 홈페이지에서 확인 가능합니다.

실시간계좌이체/ 무통장 입금
