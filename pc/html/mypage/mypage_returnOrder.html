반품절차 안내

    반품을 신청하셨습니다.

  • 반품 접수

    접수되었습니다.

  • 반품 대기

    상품이 회수 중이거나
    상품을 확인 중입니다.

  • 반품 완료

    반품된 상품확인 후 결제취소
    또는 환불을 해드립니다.

  • 반품 배송비는 상품하자나 불량으로 귀책사유가 당사에 있을 경우를 제외하고 고객변심일 경우 고객님께서 부담하셔야 합니다.
  • 환불은 현금 결제일 경우 제품 회수 확인 후 3일 이내에 환불처리 됩니다.
  • 신용카드 결제일 경우 제품 회수 확인 후 해당카드사로 승인취소가 넘어갑니다. 카드사별 취소 반영이 차이가 있습니다.
  • 각 카드사 홈페이지에서 확인 가능합니다.

반품이 불가능한 경우

  • 제품이 사용되었거나 훼손된 경우(단, 내용 확인을 위한 포장 개봉의 경우는 예외)
  • 제품에 부착되어있는 상품 택(TAG)을 제거 또는 상품 개봉으로 상품 가치가 훼손된 경우 12개월 유지됩니다.
  • 상품설명에 기재한 사용시 주의사항을 지키지 않은 경우

결제수단별 환불안내

신용카드

  • 승인취소 : 신용카드 승인취소는 매입이 되기 전이기 때문에 승인한 당일 취소됩니다.
  • 매입취소 : 카드사로 매입취소 신청을 하면 카드사에서 취소처리를 하는데 최대 1~2주 정도 소요 됩니다.
    • 카드사별 취소 반영이 차이가 있습니다.
    • 각 카드사 홈페이지에서 확인 가능합니다.

실시간계좌이체/ 무통장 입금

  • 영업일 1~3일 이내에 등록된 환불계좌로 환불됩니다.

네이버페이

  • 간편 신용카드/체크카드 : 취소 완료 후 3~5영업일 이후 환불(승인/매입 구분 불가)
  • 간편 계좌이체 : 취소 완료 즉시 환불 (단, 은행 정기점검시간 등에는 환불 불가)
  • 포인트 : 취소 완료 즉시 환불
  • 은행 점검 관련
    정기 점검 시간 : 23시 30분 ~ 00시 30분
    추가 점검 시간: 정기 점검 시간 외에 은행별 추가 점검 시간에는 해당 은행을 이용하기 어렵습니다.
    • 우리은행 : 매월 두 번째 토요일 23시 30분 ~ 일요일 06시까지
    • 농협은행 : 매월 세 번째 일요일 23시 30분 ~ 월요일 04시까지

카카오페이

  • 신용카드 : 취소완료일로부터 영업일 기준 3일~5일 이내 카드 승인 취소
  • 실시간계좌이체 : 취소완료일로부터 영업일 기준 2일~3일 이내 최초 결제 시 이용한 계좌로 입금

페이코
